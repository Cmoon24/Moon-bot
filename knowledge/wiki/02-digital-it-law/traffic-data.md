# Traffic Data (ข้อมูลจราจรทางคอมพิวเตอร์)

Traffic data refers to the metadata associated with computer communications, excluding the actual content of the messages.

## Scope of Data
Under the [[computer-crime-act-summary]], traffic data includes:
- **Source and Destination:** IP addresses, phone numbers, or user IDs.
- **Route:** The path the data took through the network.
- **Time and Duration:** When the communication started and ended.
- **Volume:** The amount of data transmitted.
- **Type of Service:** E.g., web browsing, email, file transfer.

## Retention Requirements
Service providers are legally required to retain this data for at least **90 days**. This period can be extended to **2 years** by official order if necessary for an investigation.

## Importance
Traffic data is a critical component of [[digital-evidence]], allowing investigators to trace the origin of illegal activities even if the perpetrator has deleted the original messages.
